= TV

Ispisuje sto je na TV-u, sa tvprofil.net, ili trazi navedeni string u TV programu

== Installation

  $ curl http://github.com/bkrsta/whatson/raw/master/tv.rb > ~/bin/tv

== Usage

Nakon prilagodbe kanala u `chlist` u tv.rb-u:

  $ tv [search string]

== Requirements

Ruby, RubyGems, Hpricot

  $ gem i hpricot
